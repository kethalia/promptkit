# Agent Skills

Downloadable skill packs following the [Agent Skills](https://github.com/anthropics/agent-skills) open standard. Each skill bundles prompts with reference material for deeper AI assistant capabilities.

## Programmatic Access

| Endpoint | Description |
|----------|-------------|
| [`/api/skills`](/api/skills) | JSON catalog of all skills |
| [`/api/skills/all`](/api/skills/all) | All skills with full content |
| [`/api/skills/all/download`](/api/skills/all/download) | Download all `.skill` files as zip |
| `/api/skills/{slug}` | Single skill content |
| `/api/skills/{slug}/download` | Download a single `.skill` file |
