# PromptKit

A curated collection of reusable prompts and agent skills for AI coding assistants.

---

## Quick Start

Copy the prompt content and paste it into your AI assistant, or reference the file path directly if your tool supports it.

```bash
# Access a prompt via the API
curl /api/prompts/review/pr-review

# Fetch the full LLM context file
curl /llms-full.txt
```

## AI / LLM Access

This site is optimized for programmatic AI access:

| Endpoint | Description |
|----------|-------------|
| [`/llms.txt`](/llms.txt) | Site overview following the [llms.txt spec](https://llmstxt.org/) |
| [`/llms-full.txt`](/llms-full.txt) | All prompt content in a single file for LLM context |
| [`/api/prompts`](/api/prompts) | JSON catalog of all prompts with metadata |
| `/api/prompts/{category}/{slug}` | Raw markdown for a specific prompt |
| [`/api/skills`](/api/skills) | JSON catalog of all agent skills |
